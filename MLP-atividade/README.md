MLP-ATIVIDADE

Atividade sobre criar um Perceptron de Multicamada (MLP) do zero em Python.
Primeiro importamos a biblioteca NumPy e definimos a classe MLP, onde criamos as funções de ativação, derivada e o método de treinamento. Depois inicializamos os pesos e biases com valores aleatórios e usamos laços de repetição para ajustar esses valores durante o aprendizado.
O algoritmo usa a função sigmoid para calcular as ativações dos neurônios e o método backpropagation para corrigir os erros e atualizar os pesos a cada época.
Por fim, testamos o modelo com a tabela verdade do XOR, mostrando que a rede conseguiu aprender corretamente o comportamento lógico esperado
